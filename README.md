# Resume Generator Web Application

A professional resume builder built with React (Vite) and Express.

## Structure

- `client/`: Frontend application (React + TailwindCSS)
- `server/`: Backend API (Express.js)

## Getting Started

1. **Client**: `cd client` -> `npm install` -> `npm run dev`
2. **Server**: `cd server` -> `npm install` -> `npm start`
